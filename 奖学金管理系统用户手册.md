# 高校奖学金管理系统用户手册

## 1. 系统概述

### 1.1 系统简介
高校奖学金管理系统是一款专为高校设计的奖学金管理平台，旨在简化奖学金申请、评审和管理流程，提高工作效率，确保奖学金评定的公平、公正和公开。

### 1.2 系统功能
- **学生端**：个人信息管理、奖学金申请、申请状态查询
- **管理员端**：学生管理、奖学金管理、评审管理、公告管理

### 1.3 适用人群
- **学生**：申请奖学金、查看申请状态
- **管理员**：管理奖学金、评审申请、发布公告

## 2. 系统登录与退出

### 2.1 系统访问
打开浏览器，在地址栏输入系统URL，进入系统登录页面。

### 2.2 登录系统
1. 在登录页面输入用户名和密码
2. 点击"登录"按钮
3. 根据用户角色，系统将自动跳转到相应的首页

#### 学生登录
成功登录后，学生将进入学生仪表板，显示个人信息、GPA统计和奖学金申请相关功能。

#### 管理员登录
成功登录后，管理员将进入管理员仪表盘，显示系统概览、待处理申请和管理功能入口。

### 2.3 退出系统
1. 点击页面右上角的"退出"按钮
2. 系统将自动清除登录状态，并返回登录页面

## 3. 学生用户操作指南

### 3.1 个人信息管理

#### 3.1.1 查看个人资料
1. 登录系统后，点击左侧菜单的"个人资料"
2. 页面将显示您的基本信息、学业记录统计（GPA平均分、总学分等）

#### 3.1.2 修改密码
1. 登录系统后，点击左侧菜单的"修改密码"
2. 输入原密码和新密码
3. 再次输入新密码进行确认
4. 点击"保存"按钮
5. 密码修改成功后，系统将显示提示信息

**注意事项**：
- 新密码必须包含至少6个字符
- 新密码应包含字母、数字和特殊字符，以提高安全性
- 请妥善保管您的密码，避免与他人共享

### 3.2 奖学金申请

#### 3.2.1 查看可申请奖学金
1. 登录系统后，点击左侧菜单的"奖学金申请"
2. 页面将显示所有可申请的奖学金项目列表
3. 点击奖学金名称，可查看奖学金的详细信息

#### 3.2.2 提交奖学金申请
1. 在奖学金列表页面，选择您要申请的奖学金
2. 点击"申请"按钮
3. 填写申请表单，包括个人声明、上传必要的证明材料
4. 点击"提交"按钮
5. 申请提交成功后，系统将显示提示信息

**注意事项**：
- 每位学生最多只能提交2个非拒绝状态的奖学金申请
- 同一学年内，不能重复申请同一类型的奖学金

#### 3.2.3 查询申请状态
1. 登录系统后，点击左侧菜单的"申请记录"
2. 页面将显示您所有的奖学金申请记录
3. 每条记录将显示申请的奖学金名称、申请时间和当前状态
4. 点击"查看详情"按钮，可查看申请的详细信息和评审结果

### 3.3 公告查看
1. 登录系统后，点击左侧菜单的"公告通知"
2. 页面将显示所有公告列表
3. 点击公告标题，可查看公告的详细内容

## 4. 管理员用户操作指南

### 4.1 系统概览
登录系统后，管理员将进入系统概览页面，显示以下信息：
- 待处理的奖学金申请数量
- 系统用户统计
- 最新发布的公告

### 4.2 学生管理

#### 4.2.1 查看学生列表
1. 点击左侧菜单的"学生管理"
2. 页面将显示所有学生用户列表
3. 可使用搜索框按学生姓名或学号进行筛选

#### 4.2.2 重置学生密码
1. 在学生列表页面，找到需要重置密码的学生
2. 点击"重置密码"按钮
3. 系统将生成一个临时密码，并显示在页面上
4. 告知学生临时密码，学生登录后需立即修改密码

### 4.3 奖学金管理

#### 4.3.1 查看奖学金列表
1. 点击左侧菜单的"奖学金管理"
2. 页面将显示所有奖学金类型列表

### 4.4 奖学金评审

#### 4.4.1 查看待评审申请
1. 点击左侧菜单的"奖学金评审"
2. 页面将显示所有待评审的奖学金申请列表

#### 4.4.2 评审奖学金申请
1. 在申请列表页面，点击"查看详情"按钮，查看申请的详细信息
2. 仔细阅读申请材料和相关证明
3. 点击"批准"或"拒绝"按钮
4. （可选）填写评审意见
5. 点击"提交"按钮

#### 4.4.3 查看评审历史
1. 在申请列表页面，选择已评审的申请
2. 点击"查看详情"按钮
3. 页面将显示评审结果和评审意见

### 4.5 公告管理

#### 4.5.1 查看公告列表
1. 点击左侧菜单的"公告管理"
2. 页面将显示所有已发布的公告列表

#### 4.5.2 发布公告
1. 在公告列表页面，点击"发布公告"
2. 填写公告标题和内容
3. 点击"发布"按钮

## 5. 常见问题与解答

### 5.1 登录问题
**Q: 忘记密码怎么办？**
A: 请联系系统管理员重置密码。

**Q: 登录失败显示"用户名或密码错误"怎么办？**
A: 请检查用户名和密码是否正确，注意区分大小写。如果问题仍然存在，请联系管理员。

### 5.2 奖学金申请问题
**Q: 如何知道我是否符合奖学金申请条件？**
A: 在奖学金详情页面，将显示该奖学金的申请条件，请仔细阅读。

**Q: 如何查看我的申请状态？**
A: 登录系统后，点击"申请记录"，可查看所有申请的当前状态。

### 5.3 其他问题
**Q: 系统出现异常怎么办？**
A: 请记录异常现象，联系系统管理员或技术支持。

**Q: 如何更新个人信息？**
A: 学生个人信息由管理员统一管理，请联系辅导员或管理员进行修改。
